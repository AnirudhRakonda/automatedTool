# SPA Mapper (JavaScript-Aware Web Crawler)

A lightweight CLI tool to discover internal routes of **JavaScript-heavy Single Page Applications (SPAs)** such as:
- OWASP Juice Shop
- React apps
- Angular apps
- Vue apps

This tool uses **Playwright** to render JavaScript and extract internal links.
